RagnarWM

Minimal, flexibal & userfriendly.

An Overview

Ragnar is a window manager for X written entirely in C. Codewise, the project is very minimal to avoid bugs and messy things. Ragnar uses Xlib for it's X-Server interface. Currently (v1.2) the project has about 2.1k lines of code.

Installation

There are some installation scripts for a few distros like ubuntu & arch. Those make the process of installing very easy & automated.

The Arch Script

Installing Ragnar with the arch-installation script is as easy as:

cd ~/ sh -c "$(curl -fsSL https://raw.githubusercontent.com/cococry/Ragnar/main/install/arch_installation.sh)"

The Ubuntu Script

Just run the script on your Ubuntu mashine and you're good to go:

cd ~/ sh -c "$(curl -fsSL https://raw.githubusercontent.com/suleyman-kaya/Ragnar/main/ubuntu_installation.sh)"

Installation without a Script

Installing Ragnar without a script is also a fairly easy process. Just follow these easy steps:

Installing Dependencies

Ragnar requires a few dependencies to be installed on your system. Those are:

  • xorg
  • xorg-xinit
  • git
  • gcc
  • make
  • sudo

On Arch

Install these packages with pacman:

sudo pacman -S xorg xorg-xinit git gcc make sudo

On Debian

The dependencies are installed with apt:

sudo apt-get install xorg xorg-xinit git gcc make sudo

After installing the build dependencies to your system, the next step is to clone the Ragnar git repository

Clone the Git-Repository

To clone the repository run:

git clone --depth=1 https://github.com/cococry/Ragnar

If the repository folder is on your system, cd into it and install ragnar to your mashine with the 'make' command.

Installation with 'make'

Installing ragnar onto your mashine is as easy as:

sudo make ragnar install

This requires the make package being installed and also access to sudo. The Makefile compiles ragnar and copies it to your /usr/bin directory.

Setting up .xinitrc

Now you need to specify to use Ragnar as your X Window manager. On startup of the X-Server, ragnar should start.

To do this you need to create a .xinitrc file in your home directory:

cd ~/ touch .xinitrc

Now, add 'exec ragnar' to your .xinitrc to start ragnar on X-Startup:

echo exec ragnar >> .xinitrc

Starting Ragnar on boot

If you want to start Ragnar as your PC boots, you need to call the 'startx' command which executes .xinitrc which does 'exec ragnar'.

To do this, add an if statement that checks if the X-Server has been started yet to your shell's rc file (.zshrc, .bashrc etc.):

if [[ ! $DISPLAY && $XDG_VTNR -eq 1 ]]; then startx fi

Adding Ragnar to your sessions

If you are using a session manager you need to add Ragnar to your session, so you can use it.

To do that, copy the ragnar.desktop file in your Ragnar directory to /usr/share/xsessions:

sudo cp ragnar.desktop /usr/share/xsessions/ragnar.desktop

Conclusion

That's it, you installed Ragnar to your system and now using it as your window manager. For further explanations, tutorials & information, there are lots of them on this site. If you're having any trouble installing ragnar or general problems, feel free to sumbit an issue on GitHub.
